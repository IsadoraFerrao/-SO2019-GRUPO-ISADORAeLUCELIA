Este trabalho possui a implementação de 04 algoritmos em duas versões.

(1) Gauss
	Para rodar gauss basta entrar na pasta 01_Gauss
	Escolher a pasta da versão sequencial ou paralela
	PARA RODAR: bash script.sh
	
(2) Borwein
	Para rodar Borwein basta entrar na pasta 02_Borwein
	Escolher a pasta da versão sequencial ou paralela
	PARA RODAR: bash script.sh	

(3) Monte Carlo
	Para rodar Monte Carlo basta entrar na pasta 03_Monte_Carlo
	Escolher a pasta da versão sequencial ou paralela
	PARA RODAR: bash script.sh
	
(4) Black scholes
	Para rodar black scholes basta entrar na pasta 04_Gauss
	Escolher a pasta da versão sequencial ou paralela
	PARA RODAR: bash script.sh

Os scripts automatizados estão programados para fazerem uma média de 30 iterações para 1000000000.
